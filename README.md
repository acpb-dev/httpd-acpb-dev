# Httpd

Le gabarit pour la mission principale #02. Il contient la création initiale de 3 projets:
* __Httpd__: Le code de votre librairie.
* __Httpd.CLI__: L'application console qui utilisera votre librairie.
* __Httpd.Tests__: Les tests de votre librairie.

Vous devez ajouter les références nécessaires aux différents projets. Les projets utilisent .NET 6.

À noter que cette solution utilise une arboresence de dossiers différente (src, test, etc.) pour vous montrer un autre exemple de comment structurer votre code.
